# Janani – AI-Powered Maternal Care & Crisis Response System

Janani is an AI-powered maternal health application designed to provide continuous care and rapid crisis response during pregnancy. It combines intelligent health monitoring, personalized nutrition, and emergency alert systems to ensure timely and informed medical assistance.

---

## Problem Statement

Maternal emergencies require immediate response, but delays often occur due to lack of accessible medical information and communication gaps. Existing solutions focus on tracking rather than real-time assistance.

---

##  Solution

Janani provides a **continuous care ecosystem** that:
- Understands user health through AI-powered journaling
- Offers personalized nutrition recommendations
- Stores medical records securely
- Sends context-aware emergency alerts during crises

---

##  Key Features

-  **Voice/Text Journal (English + Kannada)**  
  Users can log daily health updates via voice or text in regional and global languages.

-  **AI-Based Mood & Health Analysis**  
  Uses Gemini API to analyze emotional and physical conditions in real time.

-  **Personalized Meal Plans**  
  Generates customized South Indian meal plans with recipe suggestions based on user input.

-  **Medical Report Storage**  
  Securely stores prescriptions, ultrasound scans, and reports for quick access.

-  **Pregnancy Tracking Dashboard**  
  Provides weekly insights and guidance throughout pregnancy.

-  **One-Click SOS Alert System**  
  Sends emergency alerts along with a **medical snapshot** to doctors and emergency contacts.

---

##  Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js  
- **Database:** Firebase Firestore  
- **Authentication:** Firebase Authentication  
- **AI:** Gemini 1.5 Flash API  
- **Communication:** Twilio API (WhatsApp Alerts)  
- **Deployment:** Google Cloud Run  

---

##  System Flow

1. User logs daily health via journal  
2. AI analyzes mood and condition  
3. Personalized meal plans are generated  
4. Medical records are stored securely  
5. In emergencies, SOS sends a medical snapshot to contacts
